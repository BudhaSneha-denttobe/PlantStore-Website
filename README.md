# PlantStore-Website
live website link:https://budhasneha-denttobe.github.io/PlantStore-Website/
PlantStore Website is an interactive and user-friendly e-commerce website designed for plant enthusiasts. It allows users to browse, explore, and purchase plants, seeds, and planting accessories. The website is visually appealing, responsive, and provides a seamless shopping experience.

PlantStore Website
Features

Home Page

Welcoming landing page with featured products.

Navigation menu to access different sections of the store.

Plants Page

Displays a variety of plants.

Each plant has images, description, and price.

Users can add plants to the cart.

Seeds Page

Showcases different types of seeds.

Includes images, details, and prices.

Add-to-cart functionality for each seed.

Accessories Page

Includes planting accessories like pots, watering tools, gloves, decorative items, and other tools.

Each accessory has images, descriptions, prices, and add-to-cart buttons.

Separate pages for each accessory type.

Cart Page

Displays all selected items with quantity and price.

Calculates total cost.

Option to place order.

Responsive Design

Works well on desktop and mobile devices.

Simple, clean, and elegant layout.

Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and layout design

JavaScript – Interactive elements

Bootstrap – Responsive design and UI components

How to Use

Clone or download the repository.

Open index.html in any web browser.

Navigate through Plants, Seeds, Accessories, and Cart pages.

Click ‘Add to Cart’ on products to simulate shopping.

View total cost and place an order from the Cart page
